## 라우트 목록

### 인증 플로우

| 경로 | 설명 | 접근 |
|------|------|------|
| `/` | 랜딩 — 서비스 소개 + 시작하기 | 비로그인 |
| `/login` | Google OAuth · 이메일 로그인 | 비로그인 |
| `/signup` | 이메일 회원가입 | 비로그인 |

### 온보딩 (최초 1회)

| 경로 | 설명 | 접근 |
|------|------|------|
| `/onboarding` | 온보딩 진입점 | 로그인 + 커플 미연결 |
| `/onboarding/create` | 초대 코드 생성 (issuer) | 로그인 + 커플 미연결 |
| `/onboarding/join` | 초대 코드 입력 (receiver) | 로그인 + 커플 미연결 |

### 메인 앱 (바텀 탭)

| 경로 | 설명 | 접근 |
|------|------|------|

| `/tickets` | 전체 티켓 목록 · 상태별 필터 | 로그인 + 커플 연결 |
| `/tickets/[id]` | 티켓 상세 · 요청 이력 | 로그인 + 커플 연결 |
| `/logs` | 사용 기록 타임라인 | 로그인 + 커플 연결 |
| `/stats` | 월별 통계 · 승인율 **(V2)** | 로그인 + 커플 연결 |

### 설정

| 경로 | 설명 | 접근 |
|------|------|------|
| `/settings` | 앱 전체 설정 진입점 | 로그인 |
| `/settings/rules` | 타임아웃 · 자동발급 룰 | **issuer only** |
| `/settings/profile` | 이름 · 아바타 변경 | 로그인 |
| `/settings/notifications` | Web Push 구독 관리 | 로그인 |
| `/settings/couple` | 파트너 정보 · 연결 해제 | 로그인 + 커플 연결 |

### 모달 / 바텀시트

| 컴포넌트 | 진입 경로 | 대상 |
|----------|-----------|------|
| 티켓 사용 요청 바텀시트 | `/home` CTA 버튼 | receiver |
| 승인/거절 바텀시트 | `/home` 파트너 요청 카드 | issuer |
| 수동 티켓 발급 모달 | `/tickets` 우상단 버튼 | issuer only |

## 라우팅 규칙

### 미들웨어 처리
```
비로그인 → /login 리다이렉트
로그인 + 커플 미연결 → /onboarding 리다이렉트
로그인 + 커플 연결 → 정상 접근
```

### 역할 기반 접근 제어
- `/settings/rules` — issuer만 접근, receiver 접근 시 `/settings` 리다이렉트
- 수동 발급 모달 — issuer만 UI 노출
- 요청/승인 바텀시트 — 역할에 따라 다른 UI 렌더링

---

## 티켓 사용 요청 플로우

```
receiver: 홈 → "티켓 사용 요청하기" 탭
       → 요청 바텀시트 (티켓 선택 + 메모)
       → 요청 전송 → Web Push 발송

issuer:  홈 → 파트너 요청 카드 확인
       → 승인/거절 바텀시트
       → 승인 → 티켓 차감 + 로그 기록
       → 거절 → 티켓 유지 + 거절 사유
       → (타임아웃) → rules.timeout_action 기준 자동 처리
```
