## 서비스 개요

TicketBar는 커플이 서로의 약속을 티켓 형태로 관리할 수 있는 플랫폼이다.

"일주일에 한 번 음주"와 같은 약속을 월간 티켓 할당량으로 변환하여 유연하게 사용할 수 있게 한다. 티켓 발급자(파트너)와 사용자(본인) 간의 역할이 분리되어 있으며, 자동/수동 발급, 유효기간 설정, 사용 기록 추적 기능을 제공한다.

---

## 핵심 기능 (MVP)

### 커플 연동
- 초대 코드/링크로 파트너 연결
- 발급자(issuer) / 수신자(receiver) 역할 분리
- 룸 단위로 약속 관리

### 티켓 발급 시스템
- 자동 발급 (월별/주별 주기 설정) + 수동 발급
- 발급 시 유효기간 설정 가능, 만료 시 자동 소멸

### 티켓 사용 요청 (승인 플로우)
요청 → 파트너 Push 알림 → 승인 / 거절 / 타임아웃 반환

타임아웃 처리 3가지 옵션 (파트너가 룰로 설정):
- **자동 승인** — 티켓 차감
- **자동 거절** — 티켓 유지
- **요청 반환** — 티켓 유지, 없던 일 처리 (재요청 가능)

### 사용 기록 로그
- 날짜, 메모, 승인/거절/반환 상태 기록
- 파트너도 조회 가능

### 룰 설정 (issuer 전용)
- 타임아웃 시간 (파트너 직접 설정)
- 타임아웃 처리 방식 선택
- 자동 발급 주기 및 장수 설정
